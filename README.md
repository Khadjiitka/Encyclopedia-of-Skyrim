##  Project Description

The site is designed as a multi-page encyclopedia with:

- atmospheric backgrounds and blurred fog overlays

- side navigation menus

- category-based galleries with hover animations

- sections for guilds, magic, races, maps, bestiary, and more

- consistent medieval-inspired typography

- adaptive structure for future mobile optimization

Each page aims to recreate the unique feeling of the Skyrim universe while remaining clean and user-friendly.
